# Go基础

Go语言虽然不是我的主语言，不过Go语言的简洁性还是吸引着我，这里需要对Go语言有个整体性的认识。

对于一门语言，我们需要对我们的学习进行基本的划分，每个划分侧重理解的主题都是不同的

    1. 基本概念
    2. 包，struct, interface
    3. slice，map
    4. go，channel，select
    4. 基本库

## 基本概念

基本概念主要是涉及一门语言的两个板块：
    
    1. 基本数据类型
    2. 流程控制语句

### 基本数据类型

Go的基本数据类型主要分成四大类：

    1. 布尔型
    2. 数字类型
    3. 字符串类型
    4. 派生类型

#### 布尔型

Go的布尔类型关键字是：bool，主要就两个值：true 和 false。
